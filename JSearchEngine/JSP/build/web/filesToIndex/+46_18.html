+46 18

From Wikipedia, the free encyclopedia


Area code 018 covers the area around Uppsala, Sweden. The area code stretches outwards from Uppsala to, in clockwise order, Björklinge, Gunsta, Knivsta, Ramsta and Järlåsa.
